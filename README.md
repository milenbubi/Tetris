# Tetris
.NET console project based on classic Tetris game

Game controls:

        Right Arrow - Right
        Left Arrow - Left
        Down Arrow - To Bottom
        Space - Figure rotation
        
        S - Change current figure
        P - Pause
        F - Finish the Game
